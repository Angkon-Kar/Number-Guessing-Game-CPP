# Number Guessing Game in C++ 🎮

A simple, interactive command-line Number Guessing Game written in C++. The game generates a random number between 1 and 100, and the player has to guess it within a limited number of attempts based on the chosen difficulty level.

## ✨ Features

* **Random Number Generation:** Generates a new random number every time you play.
* **Multiple Difficulty Levels:**
  * **Easy:** 10 Chances
  * **Medium:** 5 Chances
  * **Hard:** 3 Chances
* **Interactive Feedback:** Tells you if your guess is too high or too low.
* **Attempt Tracking:** Counts and displays the number of attempts it took to win.

## 🚀 How to Run the Game

To play this game on your local machine, you need to have a C++ compiler (like GCC) installed.

### 1. Clone the repository
```bash
git clone [https://github.com/your-username/Number-Guessing-Game.git](https://github.com/your-username/Number-Guessing-Game.git)
cd Number-Guessing-Game
```
*(Note: Replace your-username with your actual GitHub username and adjust the repository name if different.)*

### 2. Compile the code
Open your terminal (or PowerShell) and compile the code.cpp file:

```Bash
g++ code.cpp -o code
```
### 3. Run the executable
After successful compilation, run the game using:

Bash
# For Windows
```
.\code
```

# For Linux/macOS
```
./code
```


##🕹️ How to Play


Run the game.

Select your preferred difficulty level by entering 1, 2, or 3.

The game will prompt you to enter a guess between 1 and 100.

Keep guessing based on the "higher" or "lower" hints.

Win the game by guessing the correct number before you run out of chances!
